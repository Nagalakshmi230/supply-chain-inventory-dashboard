# Supply Chain & Inventory Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing inventory levels, demand trends, and supplier performance.

## 🛠 Tools & Technologies

* SQL
* Power BI
* Excel

## 📊 Key Features

* Inventory tracking and stock level analysis
* Stock status classification (Shortage, Optimal, Overstock)
* Demand trend analysis
* Supplier lead time evaluation

## 🔍 Key Insights

* Multiple products fall below reorder level indicating shortages
* Demand trends highlight increasing product requirements
* Lead time impacts supply chain efficiency

## 📁 Files Included

* Power BI Dashboard (.pbix)
* SQL Queries
* Dataset (sample)

## 🚀 Outcome

Improved inventory visibility and helped identify stock shortages to support better supply chain planning.
